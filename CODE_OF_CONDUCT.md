# Code of Conduct

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone,
regardless of age, body size, visible or invisible disability, ethnicity, sex characteristics,
gender identity and expression, level of experience, education, socio-economic status,
nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful and constructive
- Giving and accepting feedback gracefully
- Focusing on what is best for the community

Unacceptable behavior includes harassment, insults, personal attacks, and discriminatory language.

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior.

## Scope

This Code of Conduct applies within all project spaces and when an individual is officially representing the project.

## Reporting

Report issues to the repository maintainers via private channel/email listed in repository profile.
